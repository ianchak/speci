# `speci task`

One-shot task generation from a plan file via GitHub Copilot CLI.

## Usage

```bash
speci task -p <path> [-c] [--sleep-after] [-v]
```

## Options

| Flag                | Description                                                    |
| ------------------- | -------------------------------------------------------------- |
| `-p, --plan <path>` | Path to the plan file generated by `speci plan` **(required)** |
| `-c, --clean`       | Remove existing task files and `PROGRESS.md` before generating |
| `--sleep-after`     | Put machine to sleep after command completes                   |
| `-v, --verbose`     | Show the full prompt and agent output                          |

## Flow

1. **Validate `--plan`** — the flag is required; exits with an error if missing.
2. **Resolve path** — resolves the plan file relative to the working directory.
3. **Validate file** — checks the plan file exists and is readable.
4. **Clean (optional)** — if `--clean` is set, removes any existing task files and `PROGRESS.md`.
5. **Initialise** — loads config, runs preflight, resolves the `speci-task` agent.
6. **Dispatch agent** — runs `speci-task.agent.md` with the plan file context.
7. **Write output** — agent writes individual task files and a `PROGRESS.md` tracking file.

## Example

```bash
speci task --plan docs/auth-plan.md
```
